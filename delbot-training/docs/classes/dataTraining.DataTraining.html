Class DataTraining

Utility class to load raw datas from your computer, parse them with a Data objet and get final tensors for a model. You first need a json file that represents a Session object, with a list of files containing human or bot trajectories. Each files must be something like:

resolution:1536,864
9131.1,Pressed,717,361
9134.8,Move,717,361
9151.8,Move,717,361
[...]
10402.3,Move,722,360
10419.1,Move,718,358
10425.8,Released,717,360

were, as first line, the resolution to normalize X and Y and the following lines are timestamp,actionType,x,y.
A part of the dataset if used as training datas and the other part for testing.
To use this class, just call load then you can call nextTrainBatch and nextTestBatch.
